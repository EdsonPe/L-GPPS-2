# L-GPPS

Infraestrutura pública de memória histórica imutável para accountability institucional.

O L-GPPS registra fiscalizações cidadãs de forma permanente e segura. Após o envio, o cidadão perde o acesso ao conteúdo bruto — o sistema guarda o dado de forma protegida e gera apenas um hash de prova.

### Princípios
- Imutabilidade: uma vez registrado, não pode ser alterado
- Proteção de dados: o usuário não mantém controle do conteúdo após envio
- Validação pelo impacto: o registro ganha força quando o problema é resolvido
- Memória pública: dados agregados e anonimizados revelam padrões de omissão e resposta institucional

Este projeto busca transformar reclamações isoladas em uma memória coletiva confiável e duradoura.
