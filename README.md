# L-GPPS

**Memória Pública Imutável para Fiscalizações Cidadãs**

O L-GPPS transforma fiscalizações cidadãs em registros permanentes e auditáveis.

### Como funciona
- O cidadão envia uma ocorrência com evidências
- O sistema gera um **hash de prova** e devolve como comprovante
- Após o envio, o cidadão perde o acesso ao conteúdo bruto
- O sistema acompanha o caso até a solução
- Os dados agregados e anonimizados formam uma memória pública confiável

### Princípios principais
- **Imutabilidade** — O registro não pode ser alterado após o envio
- **Proteção de dados** — O usuário não mantém controle do conteúdo
- **Validação pelo impacto** — O registro ganha força quando o problema é resolvido
- **Memória histórica** — Cria um histórico objetivo de omissões e respostas institucionais

Este projeto busca construir uma jurisprudência social baseada em evidência real, não apenas em reclamações isoladas.
